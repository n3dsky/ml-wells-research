# ml-wells-research
# Анализ прибыльности новых скважин в регионах добычи нефти

**Проект для компании «ГлавРосГосНефть»**

## Описание проекта

Добывающая компания «ГлавРосГосНефть» предоставила данные по пробам нефти в трёх регионах (по 100 000 месторождений в каждом). Для каждого месторождения известны геологические признаки (`f0`, `f1`, `f2`) и объём запасов нефти (`product`).

**Цель проекта** — построить модель машинного обучения, которая поможет выбрать регион для бурения новых скважин с максимальной прибылью и минимальными рисками.

## Задачи исследования

1. Загрузить и подготовить данные (проверка пропусков, аномалий, типы данных).
2. Обучить модель линейной регрессии для каждого региона и оценить качество (RMSE + средний запас).
3. Рассчитать минимальный объём сырья для безубыточной разработки скважины.
4. Разработать функцию расчёта прибыли от выбранных скважин.
5. Применить технику **Bootstrap** для оценки распределения прибыли, доверительных интервалов и риска убытков.
6. Выбрать оптимальный регион на основе прибыли и рисков.

## Результаты проекта

### Рекомендуемый регион: **Регион 2**

На основе анализа данных, моделей машинного обучения и Bootstrap-оценки рисков **Регион 2** признан оптимальным для разработки новых скважин.

**Ключевые аргументы в пользу выбора:**

1. **Минимальные риски убытков**  
   - Вероятность убыточности: **менее 2.5%**  
   - Единственный регион, удовлетворяющий порогу приемлемого риска компании.

2. **Высокая предсказуемость запасов**  
   - Модель показывает стабильные и надёжные прогнозы.

3. **Высокий потенциал прибыли**  
   - **Средняя ожидаемая прибыль:** 467 058 581.44 руб.  
   - **95% доверительный интервал:** от 59 млн руб. до 878 млн руб.

**Отклонённые регионы:**  
Регионы 1 и 3 показали низкую точность моделей и высокие риски убытков, поэтому были исключены.

**Заключительная рекомендация:**  
Начать разработку скважин в **Регионе 2** — он обеспечивает лучшее сочетание низкого риска (<2.5 %), высокой предсказуемости и значительного потенциала прибыли.
